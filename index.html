CSS: The Box Model Cheat Sheet

The Box Model

"Box model" is the term used to describe design and layout in CSS
Consisting mainly in those elements

  • Margin
  • Bording
  • Padding
  • the actual content
>
Selectors Exemples Explanation
Content Can be anything, such as a img or text The content of the box, where text and images appear
Padding padding: 20px; Clears an area around the content, before the set margin. The padding is transparent
Border border: 10px solid blue A border that goes around the padding and content, can be set in different ways with Style
Margin margin: 200px auto; Area outside the border, to set distance between screenborder or other elements. The margin is transparent
Calculus of the content When setting the size of the content,
it's needed to calculate the whole box		 320px (width)
+ 20px (left + right padding)
+ 10px (left + right border)
+ 0px (left + right margin)
= 350px

It's a basic wrap around content of boxing, to master it, it's necessary to practice!
